# EURUSD Swing Bias Bot

This repository contains a Python bias engine and an MQL5 expert advisor for trading EURUSD swings.

## Quick Start

1. Configure `.env` and `config/config.local.yaml`.
2. Run the Python bias engine once to generate a bias JSON file.
3. Place the output in the MT5 `MQL5/Files` directory for the EA to consume.

See the documentation in `docs/` for detailed instructions.
